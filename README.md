# M3 — E-Commerce Maquillaje Luxury

## Descripción
Proyecto de e-commerce básico con temática de **tienda de maquillaje luxury**.  
Incluye un carrito de compras desarrollado en JavaScript y un sistema de autenticación simple mediante modal.

Este trabajo forma parte del **Portafolio del Módulo 3**, enfocado en reforzar la manipulación del DOM, lógica de arrays y eventos.

## Tecnologías utilizadas
- HTML5  
- CSS3  
- JavaScript  
- Bootstrap 5 (CDN)

## Cómo ejecutar el proyecto
- Abrir el archivo `index.html` en el navegador  
- (Opcional) Utilizar Live Server desde VS Code
  
## Funcionalidades principales
- Visualización de productos de maquillaje
- Carrito de compras:
  - Agregar productos
  - Eliminar productos
  - Cálculo de total
- Aplicación de descuento mediante código promocional (`DESCUENTO10`)
- Autenticación básica:
  - Modal de login con contraseña maestra
- Renderizado dinámico del carrito en tabla Bootstrap
